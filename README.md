# Student Performance Prediction

End-to-end workflow for predicting student pass/fail risk using the combined UCI math + Portuguese datasets, with preprocessing, modeling, evaluation, prediction/risk scoring, and a Streamlit dashboard.

## Project Structure
```
student-performance-prediction/
├── data
│   ├── raw/           # original UCI CSVs
│   └── processed/     # cleaned + combined data, predictions
├── notebooks/         # EDA, training, evaluation notebooks
├── src/               # prediction utilities
├── models/            # saved pipelines
├── streamlit_app.py   # dashboard
└── requirements.txt
```

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Data
- Raw: place `student-mat.csv` and `student-por.csv` in `data/raw/` (already downloaded).
- Processed:
  - `student_mat_cleaned.csv`
  - `student_por_cleaned.csv`
  - `student_all_cleaned.csv` (combined math + Portuguese)
  - `student_predictions.csv` (generated by prediction pipeline)

## Notebooks
- `01_data_loading.ipynb` — load/EDA/clean.
- `02_baseline_model.ipynb` — train combined pass/fail models and save RF pipeline.
- `03_eval_predict.ipynb` — evaluate saved model, sweep decision thresholds to boost fail recall, write predictions with risk levels, plot feature importances.

Run with:
```bash
source .venv/bin/activate
jupyter notebook
```

## Training / Prediction (CLI)
Train (from notebook or script) saves model to `models/pass_classifier_rf.joblib`.

Generate predictions + fail-risk levels (writes `data/processed/student_predictions.csv`):
```bash
source .venv/bin/activate
python src/predict.py
```

## Streamlit Dashboard
Basic metrics, risk distribution, student search, and full table.
```bash
source .venv/bin/activate
streamlit run streamlit_app.py
```
Make sure `data/processed/student_predictions.csv` exists first.

## Notes
- Target: `pass` (1 if G3 >= 10, else 0).
- Risk score = fail probability (1 - P(pass)); risk buckets: Low (<0.3), Medium (<0.6), High (>=0.6).
- Current model: RandomForestClassifier trained on combined dataset (class_weight balanced).
- To boost fail recall or tighten to G2, adjust decision threshold in `03_eval_predict.ipynb` (threshold sweep cell) or retrain with higher fail class weight.
